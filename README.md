# 🚀 Firebase Database Todo App (React + Vite)

A modern and responsive **Todo Application** built using **React + Vite** and powered by **Firebase Realtime Database / Firestore**.  
This app allows users to create, update, delete, and manage tasks in real-time with a clean and user-friendly interface.

---

## 📌 Features

- 🔐 Firebase Authentication (Optional / If Enabled)
- 📝 Add, Edit & Delete Todos
- 🔄 Real-time Database Sync
- ⚡ Fast Performance with Vite
- 📱 Fully Responsive Design
- 🎨 Clean & Modern UI
- 🌐 Cloud-based Data Storage (Firebase)

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Vite, HTML5, CSS3, JavaScript  
- **Backend / Database:** Firebase (Realtime Database / Firestore)  
- **Styling:** CSS / Tailwind / Bootstrap (as per your setup)  
- **Hosting:** Firebase Hosting / Vercel / Netlify

## ⚙️ Environment Variables

Create a `.env` file in the root directory and add your Firebase credentials:

```
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

## 🚀 Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/your-username/firebase-todo-app.git
```

### 2️⃣ Navigate to Project Folder
```bash
cd pr-12-firebase-database-todoapp
```

### 3️⃣ Install Dependencies
```bash
yarn install
# or
npm install
```

### 4️⃣ Start Development Server
```bash
yarn dev
# or
npm run dev
```

App will run on:
```
http://localhost:5173
```

## 🔥 Firebase Setup Guide

1. Go to: https://firebase.google.com  
2. Create a new project  
3. Enable **Authentication** (if required)  
4. Enable **Realtime Database / Firestore**  
5. Copy Firebase config into `.env` file
